1) What is the definition of server port?

Answer: A port is a logical construct that identifies a specific process or a type of network service. A port is identified for each transport protocol and address combination by a 16-bit unsigned number, known as the port number.


2) What is server port range?

Answer: Port numbers can run from 0 to 65353. Port numbers from 0 to 1023 are reserved for common TCP/IP applications and are called well-known ports.


3) How many bits does number above represent?

Answer: Lets say I choose 8080 as my number. To represent that in bits you convert to binary which is equal to 1111110010000. Then you can convert that to bits which is 13 bits.


4) State 10 common server ports

Answer: 10 common server ports are


25 - Simple Mail Transfer Protocol (SMTP) E-mail Routing

20 - File Transfer Protocol (FTP) Data Transfer

21 - File Transfer Protocol (FTP) Command Control

22 - Secure Shell (SSH)

23 - Telnet - Remote login service, unencrypted text messages

53 - Domain Name System (DNS) service

80 - Hypertext Transfer Protocol (HTTP) used in World Wide Web

110 - Post Office Protocol (POP3) used by e-mail clients to retrieve e-mail from a server

119 - Network News Transfer Protocol (NNTP)

443 - HTTP Secure (HTTPS) HTTP over TLS/SSL
